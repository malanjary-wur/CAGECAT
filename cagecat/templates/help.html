CompArative GEne Cluster Analysis Toolbox

CAGECAT is continuously under development. If you encounter a user-interface problem, try pressing CTRL+F5 to reload all service files. If your problem still isn't resolved, feel free to submit feedback.

CAGECAT acts as a bridge between complex command line interface (CLI) executed analyses and aims to provide a user-friendly and intuitive interface to prevent difficult analyses execution for the individual researcher. Our service enables fast analysis using multiple interconnected tools, reducing the time required to execute such analyses. Researchers without programming experience encounter a steep learning curve to execute CLI analysis. Therefore, CAGECAT does not require any programming at all, lowering the technical barrier for researchers to execute gene cluster analyses and prevent the time-consuming process of learning how to execute such analyses by the CLI. Additionally, connecting multiple gene cluster analyses tools on CAGECAT creates an online data analysis pipeline without the difficulties of connecting multiple individual tools, utilizing the specialty of each tool.
